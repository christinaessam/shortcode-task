## display-posts Shortcode **Purpose:** Displays a list of recent posts based on specified categories and number of posts. **Usage:** HTML [display-posts categories="category_id1,category_id2,..." postsnum="number_of_posts"] categories: A comma-separated list of category IDs to include. postsnum: The number of posts to display. **Example** HTML [display-posts categories="5,7" postsnum="3"]
